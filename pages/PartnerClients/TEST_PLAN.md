# TEST_PLAN: PartnerClients

**URL:** `/partnerclients`  
**Роль:** Партнёр  
**Авторизация:** Требуется партнёрский аккаунт  
**Версия плана:** v2.0 | Сессия S293

---

## Назначение страницы

Управление базой клиентов партнёра. Показывает всех участников программы лояльности (LoyaltyAccount). Позволяет просматривать историю, отзывы клиента и отправлять ему личные сообщения.

---

## Happy Path сценарии

### HP-1: Список клиентов
1. Войти как партнёр с клиентами в системе лояльности
2. Открыть `/partnerclients`
3. ✅ Сетка карточек клиентов (Клиент #N)
4. ✅ Каждая карточка: номер, кол-во визитов, дата последнего визита, баланс, потрачено

### HP-2: Поиск клиента
1. Ввести номер клиента в поле поиска
2. ✅ Список фильтруется по номеру (поиск работает по числу, не по имени)
3. Очистить поле
4. ✅ Полный список возвращается

### HP-3: Просмотр детали клиента (Sheet)
1. Нажать на карточку клиента
2. ✅ Открывается боковая панель (Sheet) с:
   - Баланс баллов (с правильным склонением: Балл/Балла/Баллов)
   - Количество визитов
   - Всего начислено / потрачено
   - Дата первого и последнего визита
3. ✅ История транзакций (от новых к старым), формат даты dd.MM.yyyy
4. ✅ Отзывы клиента со звёздами и комментариями
5. Закрыть Sheet (нажать вне или кнопку закрыть)
6. ✅ Sheet закрывается, состояние списка сохранено

### HP-4: Отправка сообщения через иконку на карточке
1. Нажать иконку Mail на карточке (НЕ нажимать саму карточку)
2. ✅ Открывается диалог отправки сообщения, Sheet НЕ открывается
3. Заполнить тему и текст
4. Нажать «Отправить»
5. ✅ Toast «Сообщение отправлено», диалог закрывается

### HP-5: Отправка сообщения через Sheet
1. Открыть Sheet клиента (нажать на карточку)
2. Нажать кнопку «Отправить сообщение» внутри Sheet
3. ✅ Открывается диалог поверх Sheet
4. Заполнить и отправить
5. ✅ Диалог закрывается, Sheet остаётся открытым

## Edge Cases

| # | Сценарий | Ожидаемое |
|---|----------|-----------|
| E1 | Нет клиентов | Карточка «Нет клиентов» (пустое состояние) |
| E2 | Поиск без результатов | «Клиенты не найдены» |
| E3 | Клиент без транзакций | «История пуста» в Sheet |
| E4 | Клиент без отзывов | «Отзывов нет» в Sheet |
| E5 | Баланс 1 балл | «1 Балл» (не «1 Балла») |
| E6 | Баланс 21 балл | «21 Балл» (не «21 Баллов») |
| E7 | Баланс 11-14 баллов | «11 Баллов» (исключение) |
| E8 | Много клиентов (20+) | Сетка скроллируется, нет тормозов |

## Специфические проверки (из кода)

- **BUG-PC-001 (FIXED):** клик на иконку Mail на карточке открывает только диалог, НЕ Sheet — проверить, что Sheet не открывается при клике на Mail
- **BUG-PC-002 (FIXED):** дата в формате dd.MM.yyyy (не локаль-зависимый формат)
- **BUG-PC-003 (FIXED):** если Sheet закрылся, запрос транзакций не падает (optional chaining)
- **Нумерация клиентов:** порядковый номер = позиция по дате создания аккаунта (сортировка по created_at)
