# TEST_PLAN: PartnerSettings

**URL:** `/partnersettings`  
**Роль:** Партнёр  
**Авторизация:** Требуется партнёрский аккаунт  
**Версия плана:** v2.0 | Сессия S293

---

## Назначение страницы

Центральная страница настроек ресторана. Включает: название/описание/логотип, языки и валюта меню, каналы продаж (зал/навынос/доставка), часы работы, контакты (также видны в /partnercontacts), WiFi-пароль и дополнительные настройки.

---

## Happy Path сценарии

### HP-1: Обновить название ресторана
1. Открыть `/partnersettings`
2. ✅ Форма с полями: название, описание, логотип
3. Изменить название
4. Нажать «Сохранить»
5. ✅ Toast «Сохранено», данные обновились

### HP-2: Загрузить логотип
1. Нажать «Загрузить логотип» (ImageUploader)
2. Выбрать файл изображения
3. ✅ Превью логотипа появляется
4. Сохранить
5. ✅ Логотип сохранён, отображается в настройках

### HP-3: Настроить языки меню
1. Раздел «Языки»
2. ✅ Доступны: RU, EN, KK, DE, TR, ZH, AR, ES, FR
3. Добавить/убрать язык
4. Сохранить
5. ✅ В публичном меню появился/пропал выбранный язык

### HP-4: Настроить валюту
1. Выбрать валюту из списка (KZT, USD, EUR, RUB, GBP, TRY, AED, CNY)
2. Сохранить
3. ✅ Символ валюты обновился в интерфейсе

### HP-5: Включить каналы продаж
1. Раздел «Каналы»
2. Включить «Зал» / «Навынос» / «Доставка» (чекбоксы)
3. Сохранить
4. ✅ На PartnerHome появились соответствующие каналы в статистике

### HP-6: Настроить часы работы
1. Для каждого дня недели: включить/выключить, задать время открытия/закрытия
2. Сохранить
3. ✅ Часы работы сохранены, отображаются гостям

### HP-7: Сохранить WiFi пароль
1. Ввести пароль WiFi
2. Сохранить
3. ✅ Пароль сохранён (отображается гостям в меню)

## Edge Cases

| # | Сценарий | Ожидаемое |
|---|----------|-----------|
| E1 | Название пустое | Блокировка сохранения или предупреждение |
| E2 | Выключить все языки | Минимум 1 язык должен быть активен |
| E3 | Выключить день недели | Поля времени блокируются |
| E4 | Время открытия > времени закрытия | Предупреждение или валидация |
| E5 | Логотип слишком большой | Сообщение об ограничении размера |
| E6 | Навигация по разделам настроек (если есть табы) | Несохранённые изменения — предупреждение |

## Специфические проверки

- **DEFAULT_HOURS:** дефолт пн-вс 10:00-22:00 при первом открытии
- **Каналы:** «Зал» всегда доступен; pickup/delivery — опционально
- **Языки:** 9 языков, мультивыбор
- **Валюты:** 8 вариантов с символами
- **ImageUploader:** загрузка работает (не крашится на неподдерживаемых форматах)

- [ ] Сетка дней недели (7 дней) не выходит за экран
- [ ] Кнопки «Сохранить» достижимы без горизонтального скролла
