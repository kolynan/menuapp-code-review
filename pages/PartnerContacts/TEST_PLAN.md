# TEST_PLAN: PartnerContacts

**URL:** `/partnercontacts`  
**Роль:** Партнёр  
**Авторизация:** Требуется партнёрский аккаунт  
**Версия плана:** v2.0 | Сессия S293

---

## Назначение страницы

Управление контактными ссылками ресторана (телефон, WhatsApp, Instagram, сайт и др.). Контакты показываются гостям в публичном меню. Поддерживает два режима отображения: иконки и список.

---

## Happy Path сценарии

### HP-1: Просмотр контактов
1. Войти как партнёр с настроенными контактами
2. Открыть `/partnercontacts`
3. ✅ Список ссылок с иконками типов (Phone/WhatsApp/Instagram и т.д.)
4. ✅ Бейджи с цветовой кодировкой по типу
5. ✅ Активные/неактивные контакты различимы визуально

### HP-2: Добавить новый контакт
1. Нажать «+ Добавить контакт»
2. ✅ Открывается диалог с полями: тип, метка, URL, порядок, активен
3. Выбрать тип «phone», ввести номер
4. Нажать «Сохранить»
5. ✅ Контакт появляется в списке, диалог закрывается
6. ✅ Toast подтверждения

### HP-3: Редактировать контакт
1. Нажать иконку Edit на контакте
2. ✅ Диалог открывается с заполненными данными
3. Изменить метку, сохранить
4. ✅ Список обновился с новым значением

### HP-4: Удалить контакт
1. Нажать иконку Trash
2. ✅ Запрос подтверждения (диалог)
3. Подтвердить удаление
4. ✅ Контакт исчезает из списка

### HP-5: Включить/выключить контакт
1. Нажать иконку Power на контакте
2. ✅ Контакт меняет статус active/inactive
3. ✅ Визуальное различие (opacity, цвет)

### HP-6: Переключить режим отображения
1. Переключить viewMode с «icons» на «list» (или наоборот)
2. ✅ Список перестраивается в выбранный формат
3. Нажать «Сохранить настройки вида» (если есть кнопка)
4. ✅ Режим сохраняется (возврат на страницу → тот же режим)

### HP-7: Поиск/фильтр контактов
1. Ввести текст в поле поиска
2. ✅ Список фильтруется по метке или типу

## Edge Cases

| # | Сценарий | Ожидаемое |
|---|----------|-----------|
| E1 | Нет контактов | Пустое состояние с кнопкой добавить |
| E2 | Много контактов (10+) | Скролл, порядок по sort_order |
| E3 | Поиск без результата | «Ничего не найдено» |
| E4 | Тип «phone» — проверить URL-префикс | Сохраняется `tel:+77001234567` |
| E5 | Тип «whatsapp» — префикс | `https://wa.me/77001234567` |
| E6 | Тип «email» — префикс | `mailto:info@restaurant.kz` |
| E7 | Отменить удаление | Контакт остаётся в списке |

## Специфические проверки

- **Типы контактов:** phone, whatsapp, instagram, facebook, tiktok, website, email, map, custom — все иконки корректны
- **Цвета бейджей:** каждый тип имеет свой цвет (code → CSS class)
- **viewMode persistence:** значение сохраняется в PartnerContacts entity (recordViewMode)
- **sort_order:** контакты отображаются в заданном порядке

---

## Мобильный вид (390px)

- [ ] Список контактов не обрезается горизонтально
- [ ] Диалог добавления/редактирования не выходит за экран
- [ ] Иконки кликабельны на тач-экране (≥ 44px)
