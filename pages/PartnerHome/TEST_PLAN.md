# TEST_PLAN: PartnerHome

**URL:** `/partnerhome`  
**Роль:** Партнёр (ресторан)  
**Авторизация:** Требуется партнёрский аккаунт  
**Версия плана:** v2.0 | Сессия S290

---

## Назначение страницы

Главная панель управления партнёра. Показывает статус ресторана, текущие открытые столы, статистику заказов за сегодня и чеклист онбординга (первоначальная настройка).

---

## Happy Path сценарии

### HP-1: Новый ресторан — онбординг
1. Войти как партнёр без настроенного меню/столов/каналов/персонала
2. Открыть `/partnerhome`
3. ✅ Ожидается: показан блок онбординга «Настройка ресторана» с прогресс-баром 0/4
4. ✅ 4 шага: Меню, Столы, Каналы, Персонал — все в статусе «не выполнено»
5. Нажать кнопку «Добавить блюда» (шаг 1)
6. ✅ Ожидается: переход на `/menumanage`
7. Вернуться на `/partnerhome`
8. ✅ Ожидается: прогресс-бар обновился, шаг 1 помечен выполненным (зелёный)

### HP-2: Полностью настроенный ресторан — онбординг завершён
1. Войти как партнёр с меню + столами + каналами + персоналом
2. ✅ Ожидается: блок онбординга заменён зелёным баннером «Всё настроено»
3. ✅ Кнопка «Скачать QR PDF» → переход на `/partnertables`

### HP-3: Активные столы
1. Войти, когда есть открытые TableSession
2. ✅ Блок «Открытые столы» показывает количество (например, «3 открытых стола»)
3. Нажать на блок
4. ✅ Переход на `/partnertables`

### HP-4: Нет открытых столов
1. Войти, когда нет активных сессий
2. ✅ Блок показывает «Нет открытых столов»
3. ✅ Переход на `/partnertables` при нажатии

### HP-5: Статистика за сегодня
1. Войти с заказами за текущий день
2. ✅ Блок «Сегодня» показывает: количество заказов, выручка (в формате XK ₸ или X ₸)
3. ✅ Разбивка по каналам: «Зал» всегда виден
4. Если каналы настроены: ✅ видны «Навынос» и «Доставка»

### HP-6: Открыть меню гостя
1. В блоке онбординга нажать «Открыть меню для гостей»
2. ✅ Открывается новая вкладка с `/x?partner=...&mode=hall&lang=RU`

## Специфические проверки

- **Прогресс-бар:** отображает доли `w-0 / w-1/4 / w-2/4 / w-3/4 / w-full` — проверить при каждом значении 0-4
- **Пройденные шаги:** текст зачёркнут + зелёный цвет; кнопка CTA скрыта
- **Непройденные шаги:** показывает описание + кнопку CTA
- **Completed onboarding:** кнопка «Скачать QR PDF» ведёт на `/partnertables` ✅

## Preconditions & Тестовые данные

| Сценарий | Что подготовить |
|----------|----------------|
| HP-1 (онбординг) | Partner без Dish, Table, StaffAccessLink, channels_configured_at = null |
| HP-2 (завершён) | Partner с ≥1 Dish + ≥1 Table + channels_configured_at заполнен + ≥1 StaffAccessLink |
| HP-3 (активные столы) | ≥2 открытых TableSession (status='open') для партнёра |
| HP-5 (статистика) | ≥3 Order с created_at = today(), суммы: 5000 + 3500 + 1200 = 9700 ₸, каналы: 2× hall + 1× pickup |
| E4 (выручка ≥1M) | Order.total_price сумма ≥ 1 000 000 ₸ за сегодня |
