# Задача: Починить 3 регрессии в PublicMenu (/x)

Все три бага были задеплоены в RELEASE `260305-05 x`, но по результатам Quick Test S84 не работают.

## Баги для починки

### BUG-1: PM-S81-01 — Drawer pull-down (swipe-to-close) не работает
- **Страница:** x.jsx (PublicMenu) — компонент drawer / bottom sheet
- **Симптом:** Свайп вниз по drag handle (серая линия вверху drawer) не закрывает drawer
- **Ожидаемое:** Pull down → drawer сворачивается (стандарт mobile bottom sheet)
- **Решение:** Проверить touch event handlers на drag handle. Добавить onTouchStart/onTouchMove/onTouchEnd или pointer events для drag-to-close.

### BUG-2: PM-S81-03 — Кнопка «Отправить официанту» ниже экрана
- **Страница:** x.jsx — drawer в режиме В зале (mode=hall)
- **Симптом:** Кнопка «Отправить официанту» не видна без ручного скролла вниз. Находится за нижней границей экрана (y≈729, viewport=736px).
- **Ожидаемое:** Кнопка всегда видна без скролла — sticky footer внутри drawer ИЛИ drawer auto-expand до нужной высоты
- **Решение:** Кнопка должна быть в sticky footer внизу drawer, или drawer должен автоматически раскрываться до полного размера при открытии в режиме hall.
- **Замечание:** В режиме Самовывоз/Доставка drawer раскрывается на нужную высоту. Проверить почему В зале ведёт себя иначе.

### BUG-3: PM-S81-07 — Дублирование названия ресторана
- **Страница:** x.jsx — шапка публичного меню
- **Симптом:** «Manhatten Restaurant» отображается дважды: (1) как заголовок `<h1>` в самом верху, (2) в блоке с логотипом чуть ниже
- **Ожидаемое:** Название только один раз — в блоке с логотипом
- **Решение:** Убрать верхний заголовок с именем ресторана (оставить только блок логотип + название). Решение согласовано с Arman ещё в S81.
