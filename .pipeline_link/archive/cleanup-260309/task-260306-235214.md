## Правила склонения (русский)
- 1 → "Балл"
- 2, 3, 4 → "Балла"
- 5–20 → "Баллов"
- 21 → "Балл", 22–24 → "Балла", 25–30 → "Баллов" (цикл повторяется)

```javascript
function pluralPoints(n, t) {
  const abs = Math.abs(n);
  const mod10 = abs % 10;
  const mod100 = abs % 100;
  if (mod100 >= 11 && mod100 <= 19) return `${n} Баллов`;
  if (mod10 === 1) return `${n} Балл`;
  if (mod10 >= 2 && mod10 <= 4) return `${n} Балла`;
  return `${n} Баллов`;
}
```
