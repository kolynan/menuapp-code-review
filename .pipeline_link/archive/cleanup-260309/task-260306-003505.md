# Задача: Добавить isSubmitting guard на cart drawer swipe-close

## Контекст
В RELEASE `260306-00 x RELEASE.jsx` была починена функция swipe-to-close на drawer.
Checkout drawer уже имеет правильную защиту:
```jsx
if (delta > 80 && !isSubmitting) setDrawerMode(null);
```
Но cart drawer защиты не имеет:
```jsx
if (delta > 80) setDrawerMode(null);
```

## Фикс

**Файл:** `menuapp-code-review/pages/PublicMenu/260306-00 x RELEASE.jsx`

Найти cart drawer touch handler (ориентир: строка ~3314, функция вроде handleCartTouchEnd или onTouchEnd внутри cart drawer). Найти строку:
```jsx
if (delta > 80) setDrawerMode(null);
```
Изменить на:
```jsx
if (delta > 80 && !isSubmitting) setDrawerMode(null);
```

**Убедиться** что `isSubmitting` доступна в scope этого handler. Если нет — проверить откуда она берётся в checkout drawer handler и применить тот же паттерн.
