# Задача: Часы работы — Этап A (ручной toggle открыто/закрыто)

## Контекст

Реализуем **Вариант B** (GPT-рекомендация): баннер "Закрыто" показывается ТОЛЬКО когда ресторан закрыт.
Когда открыт — ничего в шапке (не шумит).

**Этап A (эта задача):** простой ручной переключатель — партнёр сам ставит "открыт/закрыт".
**Этап B (BACKLOG):** автоматическое расписание по дням/времени — не делать сейчас.

## Что нужно сделать

### 1) PartnerSettings — добавить toggle

Добавить переключатель в настройки партнёра:

**UI:**
```
Статус заведения
[ ◉ Открыто ] / [ ○ Закрыто ]   (toggle switch)
Microcopy: "Гости смогут видеть меню и делать заказы"
```

- Поле в entity: `is_open` (boolean, default: `true`)
- Сохранять при изменении (как другие настройки на странице)
- i18n ключи (использовать `t(...)`):
  - `settings.hours.status_label` — "Статус заведения"
  - `settings.hours.status_open` — "Открыто"
  - `settings.hours.status_closed` — "Закрыто"
  - `settings.hours.status_open_hint` — "Гости могут делать заказы"
  - `settings.hours.status_closed_hint` — "Заказы недоступны. Гости видят меню."

**Файл:** `pages/PartnerSettings/260305-00 partnersettings RELEASE.jsx`

**i18n ключи:**
