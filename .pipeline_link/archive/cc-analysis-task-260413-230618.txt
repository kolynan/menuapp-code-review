## Changes Made

### INFRA-001 (P0): Chain expansion functions [DONE]
Added 11 functions to `task-watcher-v3-legacy.py`:
- `parse_chain_yaml()` -- simple YAML parser (no PyYAML dependency)
- `load_chain_template()` -- reads from pipeline/chains/{name}.yaml
- `load_step_template()` -- reads from pipeline/steps/{name}.md
- `resolve_inline_source()` -- finds latest RELEASE/BASE JSX, cleans null bytes
- `build_chain_step_prompt()` -- replaces {{TASK_BODY}}, {{INLINE_SOURCE}}, {{CHAIN_ID}}, {{PAGE}}
- `group_chain_steps()` -- groups steps by group name (parallel within, sequential between)
- `_run_chain_step()` -- orchestrates single step (CC or Codex)
- `_run_cc_step()` -- launches Claude Code with stdin piping
- `_run_codex_step()` -- launches Codex via PowerShell Get-Content
- `run_chain_task()` -- main chain orchestrator (groups, threads, TG)

Integration point: `run_task()` checks `meta.get('chain_template')` after parsing frontmatter.
If present, delegates to `run_chain_task()` and returns.

### INFRA-002 (P1): TG notifications [DONE]

Chain start/end summary messages also sent.
Simple task TG format unchanged (backward compatible).
