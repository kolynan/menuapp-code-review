# Задача: Onboarding Checklist Block — PartnerHome

## Контекст

На странице `/partnerhome` сейчас есть большой пустой блок сверху когда данных нет (новый ресторан).
Нужно заменить его на онбординг-чеклист "4 шага запуска", который показывается только пока не все шаги завершены.

## Спека UI

### Показывать блок: только когда менее 4 шагов завершены
### Скрывать блок: когда все 4 шага выполнены (показать completion state)

---

## Структура блока

**Заголовок:** `partnerhome.onboarding.block_title` (🚀 Запуск MenuApp — 4 шага)
**Подзаголовок:** `partnerhome.onboarding.block_subtitle`
**Прогресс:** `partnerhome.onboarding.progress_label` с `{done}/{total}` — например "Готово: 1/4"

## 4 шага чеклиста

### Шаг 1 — Добавьте меню
- **Статус:** выполнен если `menuItems.length > 0` (есть хотя бы одно блюдо)
- **Заголовок:** `partnerhome.onboarding.step1.title`
- **Описание:** `partnerhome.onboarding.step1.desc`
- **CTA primary:** `partnerhome.onboarding.step1.cta_primary` → navigate to `/menumanage`
- **Secondary link:** `partnerhome.onboarding.step1.link_view_example` (просто скрыть если нет примера)

### Шаг 2 — Настройте столы
- **Статус:** выполнен если `tables.length > 0` (есть хотя бы один стол)
- **Заголовок:** `partnerhome.onboarding.step2.title`
- **Описание:** `partnerhome.onboarding.step2.desc`
- **CTA primary:** `partnerhome.onboarding.step2.cta_primary` → navigate to `/partnertables`
- **Secondary link:** `partnerhome.onboarding.step2.cta_download_qr_pdf` — показывать только если шаг выполнен

### Шаг 3 — Включите каналы заказов
- **Статус:** выполнен если хотя бы один канал включён (`hall_enabled || pickup_enabled || delivery_enabled`)
- **Заголовок:** `partnerhome.onboarding.step3.title`
- **Описание:** `partnerhome.onboarding.step3.desc`
- **CTA primary:** `partnerhome.onboarding.step3.cta_primary` → navigate to `/partnersettings`
- **Microcopy под CTA:** `partnerhome.onboarding.step3.micro_later`

### Шаг 4 — Добавьте персонал
- **Статус:** выполнен если `staff.length > 0` (есть хотя бы один сотрудник)
- **Заголовок:** `partnerhome.onboarding.step4.title`
- **Описание:** `partnerhome.onboarding.step4.desc`
- **CTA primary:** `partnerhome.onboarding.step4.cta_primary` → navigate to `/partnerstaffaccess`
- **Secondary link:** `partnerhome.onboarding.step4.link_copy_invite` — копирует invite URL в clipboard

## Индикаторы прогресса на каждом шаге

- `○` — шаг не начат (нет данных)
- `◐` — шаг начат, но не завершён (если применимо — например есть блюда но нет категорий; иначе просто ○/✅)
- `✅` — шаг выполнен

Если определить "частично" слишком сложно — достаточно ○ / ✅.

---

## Completion state (когда все 4 выполнены)

Убрать чеклист, показать:
- **Заголовок:** `partnerhome.onboarding.completed.title` (🎉 Меню готово — распечатайте QR)
- **CTA:** `partnerhome.onboarding.completed.cta_download_qr_pdf` → ссылка на скачивание QR PDF (или `/partnertables` если прямой ссылки нет)
